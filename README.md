# Weather App 🌤️

A responsive weather application built using HTML, CSS, and JavaScript that provides real-time weather information using the OpenWeather API.

## Features

- Search weather by city name
- Displays current temperature
- Shows weather condition
- Displays humidity and wind speed
- Dynamic weather emoji based on weather condition
- Responsive design for different screen sizes

## Weather Emoji Feature

The app uses dynamic weather emojis to represent current weather conditions.

Examples:
- ☀️ Clear weather
- ☁️ Cloudy weather
- 🌧️ Rainy weather
- ⛈️ Thunderstorm
- ❄️ Snow
- 🌫️ Mist/Fog

The emoji changes automatically according to the weather data received from the OpenWeather API.

## Technologies Used

- HTML
- CSS
- JavaScript
- OpenWeather API

## API Used

OpenWeather API is used to fetch real-time weather data such as temperature, humidity, wind speed, and weather conditions.
